# Teste-de-lan-amento-para-Sistema-de-Cinema

Funcionalidades:

Comprar ingresso
Consultar assentos
Cancelar compra

Regras:

Não pode vender o mesmo assento duas vezes.
Deve atualizar a quantidade de lugares disponíveis.
Deve liberar o assento após cancelamento.

Testes de lançamento:

✅ Compra normal

✅ Bloqueio de assento já vendido

✅ Cancelamento

✅ Atualização da disponibilidade
